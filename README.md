Chloropleth
===

Sample chloropleth using [d3](http://d3js.org/) and [topoJSON](https://github.com/mbostock/topojson) 
to show arbitrary data across some topology.  Heavily referenced [Mike Bostock](https://github.com/mbostock)
's incredible work.
